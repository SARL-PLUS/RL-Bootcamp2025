Essential resources to help you prepare for the bootcamp

Pre-Bootcamp preparation is highly recommended. These materials will help you get the most out of the sessions.

Recommended Reading

  • Foundational Text: Sutton & Barto: Reinforcement Learning: An Introduction
  • Video Lectures: David Silver's RL Course: DeepMind Lectures
  • Modern Deep RL: OpenAI Spinning Up: Introduction to Deep RL
  • New! Sergey Levine's Deep RL Course

Code Libraries

  • Gymnasium - Standard API for RL environments
  • Stable Baselines 3 - Reliable RL implementations
  • PyTorch - Deep learning framework
  • JAX - High-performance numerical computing

  • Gymnasium Examples - Environment usage examples
