Program Overview

The 2nd International Reinforcement Learning Bootcamp will feature a comprehensive three-day program of keynote speeches, interactive tutorials, hands-on workshops, and networking opportunities. Participants will gain practical knowledge and skills in reinforcement learning from leading experts in the field.

Workshop Details

All workshop sessions will be hands-on, with participants working on practical exercises. Please bring your laptop with you. We will provide detailed setup instructions before the event to ensure everyone has the necessary software installed.

Workshop Requirements

  • Personal laptop with sufficient battery life or power adapter
  • Python 3.9+ installed (instructions will be emailed)
  • Basic understanding of machine learning concepts is recommended
  • Familiarity with Python programming is required for hands-on sessions
